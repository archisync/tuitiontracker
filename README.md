# Tuition Tracker

Offline-first Tuition Tracker built with Next.js and Turso/libSQL embedded replica sync.

## Required environment variables

Create `.env.local` with:

- `USERNAME`
- `PASSWORD`
- `SESSION_KEY`
- `TURSO_DATABASE_URL`
- `TURSO_AUTH_TOKEN`

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
